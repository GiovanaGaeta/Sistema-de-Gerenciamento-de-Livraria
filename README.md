# Sistema de Gerenciamento de Livraria 

Este é um sistema desenvolvido para automatizar e gerenciar as operações diárias de uma livraria. 

-Funcionalidades
<br>Cadastro de Livros: Inclusão, edição, remoção e listagem de títulos
<br>Gestão de Clientes: Registro e controle de dados dos clientes.
*Controle de Estoque: Atualização automática da quantidade de livros disponíveis após cada venda.
*Registro de Pedidos: Histórico de Pedidos realizados por cliente.
